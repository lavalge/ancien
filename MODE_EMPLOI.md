# OK Cuisine â€” Mode d'emploi (rapide)

## 1) Demarrage
1. Ouvrir index.html dans un navigateur
2. Configurer l'etablissement (premiere utilisation)
3. Creer un utilisateur admin
4. Se connecter avec le PIN

## 2) Navigation
- Menu a gauche pour acceder aux modules
- Tableau de bord pour voir les indicateurs
- Boutons d'action rapide sur le dashboard

## 3) Saisie quotidienne
### Temperatures
- Ajouter un releve par zone
- Enregistrer les CCP (cuisson, refroidissement, remise en temp)

### Nettoyage
- Valider une zone nettoyee
- Ajouter une action si non conforme

### Receptions
- Saisir fournisseur, produit, lot, temperature

### Inventaire
- Ajouter un produit, ajuster les quantites

### Menus
- Generer ou saisir un menu journalier
- Export PDF possible

## 4) Gestion des risques
### Alertes
- Creer une alerte et la resoudre

### Allergennes
- Renseigner les plats et allergenes
- Exporter la matrice

### Tracabilite
- Generer des etiquettes
- Gerer les plats temoins

### Protocoles
- Consulter et mettre a jour les procedures

## 5) Audit et simulateur
### Audit
- Export DDPP
- Filtrer par periode

### Simulateur
- Lancer une simulation
- Modifier les reponses
- Finaliser et exporter

## 6) Conformite avancee
- Ajouter une formation
- Declarer un incident TIAC
- Gerer RGPD (consentement, export, suppression)
- Creer un PAI enfant
- Gagner en preuve avec archivage DLC

## 7) Export et sauvegarde
- Chaque module propose des exports PDF/CSV
- Configuration permet export JSON global

## 8) Mode vocal
- Activer le micro
- Dire "OK Cuisine" puis une commande

## Depannage rapide
- Page blanche: recharger et verifier la console
- Pas d'export: verifier le navigateur (Chrome recommande)
- Donnees perdues: reimporter un export JSON
